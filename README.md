Um projeto Node.js com Express, TypeScript e TypeORM pra gerenciar pedidos (orders) e itens (items).

## Tecnologias usadas

- **Node.js**
- **Express**
- **TypeScript**
- **TypeORM**
- **SQLite**

## Requisitos

- Node.js 18+ 
- npm ou yarn

## Rodando o projeto

Clone o repositório e instale as dependências:

git clone git@github.com:lauragolombieski/orderSystem.git

cd orderSystem

npm install

npm run build

npm run migrate

## Execute o Projeto

npm run dev

O servidor rodará na porta 3000: `http://localhost:3000`

## Para verificar a documentação:
`http://localhost:3000/api-docs`
